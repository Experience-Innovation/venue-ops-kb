# Event Operations

Day-of execution and the operational choreography that transforms a venue between events. This domain covers load-in/load-out, changeovers, show management, exhibitor services, event coordination, and the complex logistics of running simultaneous or back-to-back events.

## Scope

### In Scope

Event day coordination, load-in and load-out management, changeover planning and execution, show management and stage management, exhibitor services and booth allocation, event timelines and run sheets, multi-event scheduling, event staffing deployment, setup and teardown operations.

### Out of Scope

AV/production technical operations (covered in av-and-production), event booking and contracting (covered in booking-and-sales), F&B event service (covered in food-and-beverage).

## Key Functions

Event day management, changeover coordination, load-in/load-out supervision, show calling, exhibitor services, event timeline management, multi-venue scheduling, setup crew coordination, post-event evaluation.
