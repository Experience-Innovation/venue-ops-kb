# Layered Security Strategy

Multi-measure security approach where physical, technological, and procedural controls work together in concentric layers, validated by RAND Corporation research.

## Overview

A layered security strategy deploys multiple, overlapping security measures so that no single point of failure compromises venue safety. The RAND Corporation's 2025 report on improving security at soft targets and crowded places validates this approach as the most effective framework for venue security.

## Layers

1. **Intelligence and threat assessment:** Pre-event threat analysis, behavioral indicators
2. **Perimeter security:** HVM, fencing, surveillance, lighting
3. **Screening:** Weapons detection, bag check, credential verification
4. **Interior monitoring:** AI video analytics, crowd density monitoring, roving patrols
5. **Response capability:** Trained staff, law enforcement presence, emergency protocols
6. **Cyber security:** Network segmentation, POS isolation, OT protection

## CISA Framework

The CISA Venue Guide for Security Enhancements (December 2025) organizes security considerations by implementation complexity (Low/Medium/High) and cost, providing a practical roadmap for venues at different maturity levels.

## Key Principle

Security measures are not effective in isolation — they must work as an integrated system. The gap between any two measures is the vulnerability.
