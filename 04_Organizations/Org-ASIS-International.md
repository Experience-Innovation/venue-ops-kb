# ASIS International

Global security management association developing the standards framework for venue security programs. SPC.1 (Organizational Resilience) is accepted under the US DHS PS-Prep program for demonstrating security and resilience maturity. PAP standard provides the systematic approach to physical protection system design (detection, delay, response). FPSM guideline covers mechanical, human, and design security measures — emphasizing that mechanical controls must not be relied upon in isolation.
