# AVIXA Audiovisual Standards

AVIXA (Audiovisual and Integrated Experience Association) publishes standards governing AV system design, including DISCAS (Digital Signage Display Image Comparison and Assessment), ACU (Audio Coverage Uniformity), and RP-C303.01 (AV cybersecurity). These standards define performance targets for venue AV installations.
