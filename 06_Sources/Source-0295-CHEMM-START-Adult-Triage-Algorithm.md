# CHEMM/DHHS START Adult Triage Algorithm

This US government-hosted reference page provides the primary authoritative documentation of the START (Simple Triage and Rapid Treatment) triage system. START was developed in 1983 by the Newport Beach Fire and Marine Department and Hoag Hospital in Newport Beach, California, with key co-developer Victoria Cleary, EMS Manager.

The page documents the four-category triage system: Deceased/Expectant (Black), Immediate (Red), Delayed (Yellow), and Walking Wounded/Minor (Green). The assessment sequence follows Respiration, Perfusion, Mental Status (RPM). The 1996 modification by Benson et al. substituted radial pulse for capillary refill, improving accuracy in cold temperatures. START remains the most widely used mass casualty triage system in the United States.
