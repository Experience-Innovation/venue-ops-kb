# MUCC Mass Casualty Triage Instructional Guideline

This EMS.gov publication presents the Model Uniform Core Criteria (MUCC) for mass casualty triage, a federal effort to develop a national guideline for MCI triage in the United States. The MUCC framework establishes core criteria that any triage system should meet, and SALT (Sort, Assess, Lifesaving Interventions, Treatment/Transport) was designed to be fully compliant with these criteria.

START triage, while widely used, is not MUCC compliant because it does not incorporate lifesaving interventions before categorization and uses four categories rather than the five required by MUCC. SALT replaces START in jurisdictions that have adopted the MUCC framework. The document provides the basis for understanding why SALT is increasingly mandated as the recommended national standard for mass casualty triage.
