# IAVM Arenas Performance Reporting Handbook

The IAVM Arenas Performance Reporting Handbook standardizes definitions and metrics for arena operations, enabling cross-venue benchmarking. It defines the three primary deal structures for entertainment content (flat rental, co-promotion, in-house/self-promote), establishes the pro forma template for evaluating concert deals, and codifies arena-specific KPIs including Per Cap (net ancillary revenue per turnstile count), Event Yield (average net profit/loss per event), Event ROI, and Cost per Dark Day. The handbook also covers scheduling terminology (Event Day, Non-Event Day, Dark Day), changeover/conversion costs, and the financial architecture of arena concert licensing.
