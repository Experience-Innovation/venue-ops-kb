# Event Venue Contract Template - LawDepot

Legal reference resource clarifying that public assembly venue contracts are structured as licenses, not leases. The licensee is granted a non-exclusive, limited right to use designated spaces for a specified period, explicitly not creating a tenancy or leasehold interest. This distinction matters legally because a licensee has no right to possess the property and the venue retains the right to enter at any time.
