# Salesforce Data Cloud — Fan Engagement and Personalization Use Cases

Salesforce documentation covering Data Cloud (formerly Genie CDP) capabilities for fan engagement. Features include composite insights such as customer lifetime value scores and fan engagement scores combining media consumption, ticket purchases, and merchandise purchases. Supports predictive models for upgrade receptiveness, churn risk, and satisfaction forecasts, with native integration to Marketing Cloud and Commerce Cloud.
