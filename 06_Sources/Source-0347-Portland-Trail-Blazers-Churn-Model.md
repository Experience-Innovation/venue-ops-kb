# Churn and Retention from the Portland Trail Blazers

Case study covering the Portland Trail Blazers' development of a real-time season ticket holder renewal model. The system produces a single continuous renewal-probability score per customer updated throughout the season, replacing an annual batch model. The architectural shift from annual batch scoring to real-time continuous scoring enables intervention before churn intent solidifies. Model features include tenure, attendance percentage, distance from stadium, contact history, and base revenue.
