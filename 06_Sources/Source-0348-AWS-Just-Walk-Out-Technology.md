# Elevate Your Retail Experience with Just Walk Out Technology — AWS

AWS documentation covering Amazon Just Walk Out technology's proprietary edge architecture. The system processes sensor fusion, computer vision, and AI receipt generation locally using a highly concurrent asynchronous architecture that maintains reliability when connectivity is intermittent. Edge computing reduces latency to sub-10ms for critical applications versus 100-300ms for cloud round-trip, with only summarized or flagged data sent to cloud for long-term analytics.
