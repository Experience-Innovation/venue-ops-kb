# ISO 22343-1:2023 — Hostile Vehicle Mitigation Barrier Standard

International standard for testing and classifying vehicle security barriers at venue perimeters. Critical for venue security design — supersedes legacy PAS 68 and IWA 14-1, with stricter fail conditions and updated vehicle categories reflecting modern commercial vehicles.
